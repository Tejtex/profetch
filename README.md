# profetch
![license](https://img.shields.io/badge/license-MIT-blue.svg)

profetch is like neofetch but for projects


![example](example.png)

## installation

```bash
go install github.com/tejtex/profetch@latest

```

make sure `$GOPATH/bin` or `$HOME/go/bin` is in your `$PATH`

## usage

```bash
profetch
```
